These are the following PROS bindings that will exist for the shell for the initial release.
Most of the devices are missing from this set because the initial versions of the shell will
not have conditional logic or variables


motor <port(s)> <subcommand>
    <port(s)>   - A list of ports that this command will affect. This is either
                  a single number, or a tuple of numbers. The numbers should not
                  have a suffix

    <subcommand> - The action to take on the motors
        goto        - Move absolute
        move        - Move relative
